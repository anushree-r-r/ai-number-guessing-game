# AI Adaptive Number Guessing Game

This is a Python-based number guessing game with:
- Difficulty levels
- AI-based hot/cold hint system
- Adaptive difficulty based on performance
- Time tracking
- Score system

## How to Run
1. Download the file
2. Run using:
   python game.py

## Features
- Dynamic difficulty adjustment
- Intelligent hints (Hot/Cold)
- Performance tracking

## Built With
- Python
